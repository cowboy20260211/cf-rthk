# 香港电台 CF 版 - 使用说明书

**文档版本**: 1.0.0  
**编制日期**: 2026-02-22

---

## 1. 产品简介

香港电台 CF 版是一个在线收听与节目点播平台，让您可以直接在浏览器中收听香港电台的直播节目和回顾往期内容。

### 1.1 主要功能

- 📻 **直播收听** - 实时收听第一台、第二台、第五台节目
- 📋 **节目点播** - 回顾过往节目内容
- ⭐ **收藏功能** - 收藏喜欢的节目以便快速访问
- 📱 **跨设备使用** - 支持电脑、手机、平板电脑

---

## 2. 快速开始

### 2.1 访问方式

在浏览器中访问以下地址：

- **在线访问**: https://cf-rthk.pages.dev

### 2.2 页面导航

| 页面 | 路径         | 功能                   |
| ---- | ------------ | ---------------------- |
| 首页 | `/`          | 热门节目推荐、频道入口 |
| 直播 | `/live`      | 三个频道的实时广播     |
| 节目 | `/programs`  | 按频道浏览所有节目     |
| 收藏 | `/favorites` | 已收藏的节目列表       |
| 我的 | `/profile`   | 用户设置和播放历史     |

## 3. 功能使用指南

### 3.1 收听直播

#### 步骤 1: 进入直播页面

点击导航栏中的「直播」选项，或在首页点击任意频道卡片。

#### 步骤 2: 选择频道

页面显示三个可用频道：

| 频道   | 频率       | 内容类型           |
| ------ | ---------- | ------------------ |
| 第一台 | FM 92.6MHz | 新闻、财经、时事   |
| 第二台 | FM 94.8MHz | 流行音乐、青年节目 |
| 第五台 | AM 783kHz  | 文化、教育、社区   |

#### 步骤 3: 开始播放

点击频道卡片上的播放按钮即可开始收听。底部播放器栏会显示当前播放状态。

#### 播放控制

- **播放/暂停**: 点击播放按钮切换
- **音量调节**: 拖动音量滑块或点击静音图标
- **频道切换**: 返回直播页面选择其他频道

### 3.2 节目点播

#### 步骤 1: 浏览节目

点击导航栏中的「节目」选项，进入节目列表页面。

您可以：

- 按频道筛选节目
- 使用搜索功能查找特定节目

#### 步骤 2: 查看节目详情

点击任意节目卡片，进入节目详情页。

节目详情页显示：

- 节目名称和描述
- 播出时间
- 主持人信息
- 往期节目列表

#### 步骤 3: 播放往期节目

在节目详情页的往期节目列表中：

1. 选择您想收听的日期
2. 点击播放按钮开始播放

### 3.3 收藏功能

#### 添加收藏

在节目详情页，点击「收藏」按钮（星形图标）即可将节目添加到收藏列表。

#### 查看收藏

点击导航栏中的「收藏」选项，查看所有已收藏的节目。

#### 取消收藏

在收藏列表中，再次点击已收藏节目的星形图标即可取消收藏。

### 3.4 底部播放器

无论您身处哪个页面，底部播放器栏始终可见：

- 显示当前播放的频道/节目名称
- 显示播放进度（仅点播内容）
- 提供播放/暂停控制
- 提供音量控制

## 4. 常见问题

### 4.1 播放问题

#### Q: 直播无法播放怎么办？

**解决方案**:

1. 检查网络连接是否正常
2. 尝试刷新页面
3. 检查浏览器是否阻止了音频播放
4. 尝试使用其他浏览器

#### Q: 播放卡顿怎么办？

**解决方案**:

1. 检查网络速度
2. 尝试切换到较低质量的音频流
3. 关闭其他占用带宽的应用

#### Q: 节目列表加载失败怎么办？

**解决方案**:

1. 刷新页面重试
2. 检查网络连接
3. 稍后再试（RTHK 网站可能暂时不可用）

### 4.2 功能问题

#### Q: 收藏的数据会丢失吗？

**解答**: 收藏数据保存在浏览器本地存储中。清除浏览器缓存会导致数据丢失。

#### Q: 可以同时播放直播和点播吗？

**解答**: 不可以。系统会自动切换到您最新选择的内容。

### 4.3 兼容性问题

#### Q: 支持哪些浏览器？

**解答**: 推荐使用以下浏览器的最新版本：

- Chrome
- Firefox
- Safari
- Edge

#### Q: 支持手机和平板吗？

**解答**: 是的，系统采用响应式设计，支持各种屏幕尺寸。
